# Cermati Test Backend

Scraping Using cherio, axios, async.

## How to INSTALL

```
npm install
```

## How to RUN

```
npm run start
```


## How to BUILD

```
npm run build
```


## How to FORMAT

```
npm run format
```


## How to LINT

```
npm run lint
```
